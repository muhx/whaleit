    - **duplicate_detector.rs**: `detect_duplicates(candidates, existing_in_window)`:
      - Three-key gate: same `account_id` + amount within $0.01 + date within ±3 calendar days. If gate fails → no DuplicateMatch emitted for that candidate.
      - Confidence formula: `confidence = (amount_exactness*0.4 + date_closeness*0.3 + payee_similarity*0.3) * 100` with `amount_exactness = 1.0 if |Δ|<0.001 else 0.5`, `date_closeness = 1.0 - min(|Δdays|,3)/3`, `payee_similarity = strsim::normalized_levenshtein(normalize_merchant(e.payee), normalize_merchant(c.payee))`.
      - Bucket: `>=95 → ALMOST_CERTAIN`, `70-94 → LIKELY`, `50-69 → POSSIBLE`, `<50 → suppressed (NO emission)`.
      - Within-batch: if a CSV contains the same row twice, second occurrence flags first as duplicate.
